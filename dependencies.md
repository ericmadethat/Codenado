# Dependencies

# For code editor:
For vanilla ShareJS with text only:

`$ meteor add mizzao:sharejs`

For ShareJS with the Ace editor:

`$ meteor add mizzao:sharejs-ace`

Run update to update sharejs up to v.0.7.3 or it won't work:

`$ meteor update`
